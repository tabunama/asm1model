Hydraulic delay module
======================

Import
------

.. code-block:: python

   from asm1model import hyddelay

Class
-----

``HydDelay(XINIT, PAR1, T)``

Parameters
----------

- ``XINIT``: initial delay state vector
- ``PAR1``: ASM1 parameter vector used by the block interface
- ``T``: time constant

Notes
-----

This page documents the module-level interface intended for future flowsheet construction.
